# Agent Instructions

- don't use typescript ever
- don't use advanced javascript features like Set for example
- keep the code simple and readable
- unless told to, give simple solutions
- when coding functions, try to assemble them as pure functions
- use pascalCase inside code, CamelCase for class definitions and use kebab-case for file naming conventions
- when running `npm install`, always use `--save-exact`

Example naming:

- `myFunction()`
- `class MyFunction`
- `my-function.js`

Consult OPERATIONAL.md for information about the project
