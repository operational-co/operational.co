# Manage projects and teammates

Operational supports multiple projects.
Each project has its own events, categories, and API keys.

A simple setup:

1. Create a production project.
2. Create a staging project.
3. Invite teammates only to the project they need.
4. Rotate project API keys if access changes.

You can also use events to keep a clear record of project activity.
