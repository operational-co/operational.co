# Structured events

You can choose how an event is displayed with the `type` field.

## Text (default)

```js
const e = {
  name: "user upgraded",
  avatar: "💳",
  content: "Moved from Starter to Pro",
};
```

<Item client:only="vue"
  item={{
    name: "user upgraded",
    avatar: "💳",
    createdAt: "2026-02-28",
    content: "Moved from Starter to Pro"
  }}
></Item>

## JSON

```js
const e = {
  name: "new lead metadata",
  avatar: "🧩",
  type: "json",
  content: {
    source: "landing-page",
    utmCampaign: "launch-2026",
    score: 81,
  },
};
```

<Item client:only="vue"
  item={{
    name: "new lead metadata",
    avatar: "🧩",
    createdAt: "2026-02-28",
    type: "json",
    content: {
      source: "landing-page",
      utmCampaign: "launch-2026",
      score: 81
    }
  }}
></Item>

## Rows

```js
const e = {
  name: "payment failed",
  avatar: "⚠️",
  type: "rows",
  content: [
    { label: "User ID", content: "usr_123" },
    { label: "Plan", content: "Pro" },
    { label: "Amount", content: "$49.00" },
    {
      label: "Gateway response",
      type: "json",
      content: { code: "card_declined", retryable: true },
    },
  ],
};
```

<Item client:only="vue"
  item={{
    name: "payment failed",
    avatar: "⚠️",
    createdAt: "2026-02-28",
    type: "rows",
    content: [
      { label: "User ID", content: "usr_123" },
      { label: "Plan", content: "Pro" },
      { label: "Amount", content: "$49.00" },
      {
        label: "Gateway response",
        type: "json",
        content: { code: "card_declined", retryable: true }
      }
    ]
  }}
></Item>

Notes:

1. If you do not pass `type`, Operational uses `text`.
2. Do not put `rows` inside `rows`.
